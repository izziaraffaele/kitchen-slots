# KitchenSlots

This a projeect I've done during a recruitment process

## Getting started

Install project dependecies

```bash
yarn install
```

Run the CLI application with the test input file

```bash
yarn dev tests/fixtures/test-input.csv
```

Edit the test file or provide your own to test with a different set of inputs

## Production build

Compile code for production using the following command

```bash
yarn build
```

## Tests

Only integration tests are provided. To run the test suite use the command below

```bash
yarn test
```
